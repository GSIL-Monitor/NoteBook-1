# 计算属性和侦听器
## 计算属性
对于任何复杂的逻辑，都应当使用计算属性
### 基础例子
```javaScript
var vm = new Vue({
            el: 'example',
            data: {
                message: 'Hello'
            },
            computed: {
                //计算属性的getter
                reversedMessage: function () {
                    return this.message.split('').reverse().join('')
                }
            }
        })
```
如上代码，**reversedMessage**就为计算属性
